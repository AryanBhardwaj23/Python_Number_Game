A simple, interactive terminal-based game built with Python.
The computer selects a secret number between 1 and 100, and the player’s goal is to find it.

-Tells the user to "GO HIGHER" or "GO LOWER" after every guess.

-Prevents the game from crashing if the user enters text instead of a number.

-Alerts the user if their guess falls outside the 1–100 boundary.

-Keeps the game running until the correct answer is found.
